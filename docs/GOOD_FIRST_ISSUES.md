# Good First Issues

This page lists a few curated tasks that are good entry points for new
contributors. Each item should be self-contained and well-scoped.

- Improve `contrib/plugin/mailer` examples: clean up `examples/mailer_xoauth2_demo`
  and add a README explaining how to run it locally (env vars required).
- Add unit tests for `contrib/plugin/mailer/mock.go` to cover edge cases.
- Add a small example showing how to register and use a plugin in `examples/plugins`.
- Add docs for plugin lifecycle and a short example in `docs/plugins.md`.
- Add a small linter configuration or CI docs entry that explains how to run
  `golangci-lint` locally (already suggested in CONTRIBUTING.md).

If you'd like to work on one of these, claim it by opening an issue and
tagging it with `good first issue` and `help wanted` and assign it to yourself.
